Smart Bottle

Applied Technical Skills: Object Oriented Programming, Android

This project was about creating a smart water bottle. I was a team member of the software team. The main function of the water bottle is to measure and monitor the hydration of the user. We created the mobile application using Android Studio. The app interacts with the bottle using Bluetooth. When the user uses the bottle to drink water, the hardware keeps a record of the fluid level and sends the data to the mobile app. The user can then interacts with the app to check the hydration level.

We achieved the project goal within the limited time frame. This was a challenging project for me since it involved interaction with hardware using short-range technology such as Bluetooth. Anyhow I enjoyed learning something new.
